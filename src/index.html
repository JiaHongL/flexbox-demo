1.flex vs inline-flex  [html結構：1個div(.box)後面接著1個span]
flex：具有block的特性.
span 
           
              .box{
                  display: flex;
                  width: 100px;
                  height: 50px;
                  border: 1px solid black;
                  background:green;
              }
inline-flex：具有inline-block的特性.
span 
      
                .box{
                    display: inline-flex;
                    width: 100px;
                    height: 50px;
                    border: 1px solid black;
                    background:green;
                }
