# Flexbox

## 容器相關屬性 (container)

|   屬性  |   作用  |
| :------| :------|
|  display | flex佈局 |
|  flex-direction | 容器項目 排列方向 |
|  flex-wrap | 容器項目 換行方式 |
|  justify-content | 容器項目 水平排列方式 |
|  align-items | 容器項目 垂直排列方式(單行) |
|  align-content | 容器項目 垂直排列方式(多行) |

|   屬性  |   合併  |
| :------| :------|
|  flex-flow| direction + wrap|

## 項目相關屬性 (item)

|   屬性  |   作用  |
| :------| :------|
|  flex-basis | 初始大小 |
|  flex-grow | 增長比例 |
|  flex-shrink | 壓縮比例 |
|  order | 排列順序 |
|  align-self|設定本身垂直排列方式(覆蓋容器設定) |

|   屬性  |   合併  |
| :------| :------|
|  flex-flow| grow + shrink + basis|
